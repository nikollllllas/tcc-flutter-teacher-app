# tcc_attendance_app

## Getting Started

### Installation Steps

1. **Clone the repository:**

```bash
git clone https://github.com/nikollllllas/tcc-flutter-teacher-app.git
```

2. **Navigate to the project directory:**

```bash
cd tcc-flutter-teacher-app
```

3. **Install dependencies:**

```bash
flutter pub get
```

4. **Run the application:**

```bash
flutter run
```
